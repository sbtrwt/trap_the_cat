# trap-the-cat
Click grid cell and trap the cat.


## Code Implementation
- Dependency Injection
- Service locator
- Observer design pattern
- ScriptableObject
- MVC-S architecture

  
### HOW TO PLAY
- Player can click grid cell to block cat path
- To restart click restart
